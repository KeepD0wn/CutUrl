# Сайт по сокращению ссылок на ASP.Net Core
Сайт написан на фреймворке ASP.Net Core и предназначен для: 
* сокращения ссылок 
* подсчёта колличества переходов по сокращённой ссылке 
* редактирования существующих коротких ссылок
